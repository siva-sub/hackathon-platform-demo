
# Hackathon Management Platform

## About The Project

This project is a **Hackathon Management Platform**, a web application designed to streamline the organization, execution, and participation in hackathon events. It provides distinct interfaces and functionalities tailored to three key user roles: Administrators, Participants, and Judges.

The platform aims to facilitate the entire lifecycle of a hackathon. Administrators can set up events, define rules and problem statements, and manage submissions. Participants can discover events, submit their innovative projects, and track their progress. Judges can evaluate submissions based on predefined criteria across multiple rounds.

The application also includes a demonstration of how AI-powered assistance (currently using mock responses) can be integrated to provide insights and tips for administrators and judges, enhancing the overall management experience.

## Key Features

### For Administrators:
*   **Event Setup & Customization:** Define event details, rules, timelines, and customize a public landing page.
*   **Problem Statement Management:** Create and manage multiple problem statements per hackathon.
*   **Submission Question Configuration:** Design custom questions for participants' submissions.
*   **Dynamic Judging Round & Criteria Setup:** Define multiple judging rounds with specific criteria and max scores.
*   **Judge Assignment:** Assign judges to specific rounds.
*   **Submission Oversight:** View, filter, and manage all submitted projects. Update submission statuses (e.g., advance to next round, disqualify, mark as winner/runner-up).
*   **Stage Management:** Intuitively progress the hackathon through stages (accepting submissions, judging rounds, conclusion).
*   **AI-Powered Assistance (Mock):** Receive best-practice tips on hackathon management.
*   **Multi-Hackathon Management:** Create and switch between managing multiple hackathon events.

### For Participants:
*   **Event Discovery:** View public hackathon pages with details, rules, prizes, and problem statements.
*   **Project Submission:** Submit projects, addressing specific problem statements and answering custom questions.
*   **Submission Tracking:** View their submitted projects and current status.

### For Judges:
*   **Judging Dashboard:** View submissions assigned for the current judging round.
*   **Two-Column Evaluation Interface:** Efficiently view submission details on one side and input scores/feedback for criteria on the other.
*   **Score & Comment:** Provide scores and comments for each criterion and an overall summary.

### General:
*   **Persona-Based Demo:** Easily switch between Admin, Judge, Participant, and Public User roles to experience different functionalities.
*   **Multiple Demo Hackathons:** Explore pre-configured hackathons at various stages (Upcoming, Accepting Submissions, Judging, Winners Announced, Concluded).
*   **Dark Mode:** User preference for UI theme.
*   **Responsive Design:** Adapts to different screen sizes.

## Getting Started (Demo)

1.  Open the application.
2.  You'll be presented with a **Persona Selection** screen.
3.  **Public Visitor:**
    *   Click "View All Hackathon Demos" or select the "Public Visitor" persona.
    *   Browse through different demo hackathons, each at a unique stage of completion, to see how the public page adapts.
4.  **Admin, Judge, or Participant:**
    *   Select a specific persona (e.g., "Administrator", "Judge 1 (Round 1)", "Participant 1").
    *   The system will set an appropriate "active hackathon" context for these roles, often one that is relevant to their tasks (e.g., an "Accepting Submissions" hackathon for a participant).
    *   Administrators can also create new hackathons or switch the active hackathon they are managing from their dashboard.

Explore the various dashboards and forms to see the platform in action! Use the "Reset Demo & Re-initialize Data" button on the persona selection screen to revert to the initial mock data.
